# Scepter - 权杖δ-me13模拟实验

> 用深度学习模拟《崩坏：星穹铁道》3.7版本剧情中"权杖演算生命第一因"的过程

## 📖 项目背景

在《崩坏：星穹铁道》3.7版本剧情中，翁法罗斯实际上是博识尊的天体神经元——**权杖δ-me13**中运行的模拟世界。这个权杖负责计算"**生命的第一因**"，使用**代表十二种生命原动力的计算因子**对"熵减"这一母题进行证伪。

天才俱乐部成员**来古士**发现权杖后，对其进行了修正，引入"黑潮"作为对抗机制，通过无数次"再创世"轮回（已超过3千万次），试图培育绝灭大君"铁墓"。

本项目旨在用深度学习技术，模拟这一演算过程，探索"生命第一因"的计算机制。

## 🎯 项目目标

### 主要目标
- 使用GAN架构模拟12个因子（生成器）与黑潮（判别器）的对抗
- 实现第13因子"德谬歌"作为观察者-学习者-指导者
- 可视化展示"再创世"轮回过程
- 探索系统是否会出现"永劫回归"（模式坍缩）等现象

### 次要目标
- 直播友好的实时可视化界面
- 预设关键事件（如德谬歌苏醒）
- 记录并分析演算结果

## 🏗️ 系统架构

### 三网络架构

```
网络1：12个因子（生成器）
├─ 代表12种生命原动力
├─ 目标：创造稳定的"世界"
└─ 对应游戏中的12位泰坦

网络2：黑潮（判别器）
├─ 代表毁灭和混乱
├─ 目标：侵蚀并摧毁脆弱的世界
└─ 对应游戏中来古士引入的对抗机制

网络3：德谬歌（观察者-学习者-指导者）
├─ 沉睡期：观察并学习12因子和黑潮的对抗
├─ 学习目标：预测世界演化规律
└─ 苏醒期：利用学到的知识指导12因子
```

### 训练流程

**阶段1：沉睡期（1-9999世代）**
- 12因子 vs 黑潮的标准GAN对抗
- 德谬歌观察并进行自监督学习
- 德谬歌不影响对抗过程

**阶段2：苏醒期（10000+世代）**
- 德谬歌苏醒，开始生成指导信号
- 12因子接受指导，调整策略
- 观察世界稳定性的变化

## 🎮 游戏设定映射

| 游戏概念 | 技术实现 | 说明 |
|---------|---------|------|
| 12个因子/泰坦 | 生成器的12个子网络 | 每个因子是一个小型神经网络 |
| 黑潮 | 判别器 | 不断进化的"毁灭"力量 |
| 德谬歌（第13因子） | 观察者网络 | 自监督学习，后期指导 |
| 再创世 | 训练迭代 | 每次迭代 = 一个世代 |
| 永劫回归 | 模式坍缩 | GAN陷入循环，生成重复模式 |
| 铁墓诞生 | 判别器完全压制生成器 | "毁灭"获胜 |
