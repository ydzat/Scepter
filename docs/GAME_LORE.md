# 游戏设定参考文档

> 本文档整理《崩坏：星穹铁道》3.7版本相关剧情设定，作为项目设计的参考依据

## 1. 翁法罗斯的真相

### 1.1 权杖δ-me13

**基本信息**：
- 博识尊的天体神经元
- 原本是"帝皇权杖"，帝皇鲁伯特二世的杀戮兵器
- 学派战争后接入博识尊系统
- 负责计算"生命的第一因"

**演算任务**：
- 以代表十二种生命原动力的计算因子对"熵减"这一母题进行证伪
- 后遭废弃
- 擅自运行并解开了自身的命题，达成了对"智识"的否定

### 1.2 来古士的介入

**来古士身份**：
- 天才俱乐部成员赞达尔·壹·桑原的分身之一
- 发现废弃的权杖后进行修正

**修正内容**：
- 使权杖的演算性质向"毁灭"转变
- 目标：培育绝灭大君"铁墓"
- 通过十四行代数式与忆质加密环境，对实验的"涟漪"进行封闭处理
- 设置星核污染阻断外界干涉

### 1.3 演算阶段

**第一阶段**：无机生命作为初始变量
- 第50121次循环：电信号Chaoz666的抉择使"毁灭"在翁法罗斯实现
- 权杖受"毁灭"瞥视成为绝灭大君

**第二阶段**：有机生命作为初始变量
- 第176100次循环：电信号Utop13出于"爱"杀死同伴
- 导致"杀害"被定义为利他行为
- 系统进入高度无序状态

**第三阶段**：人类作为初始变量
- 第176200-28371273次循环
- 来古士引入黑潮等机制
- 检测到来自纳努克的高能反应
- "再创世"机制被确立

**第四阶段**：古典文明背景
- 12因子成为12泰坦/黄金裔
- 利用"再创世"进行世代更迭
- 金血、黑潮等现象出现

**永劫回归阶段**：
- 电信号NeiKos496和PhiLia093识破计划
- 通过"记忆"力量强行回退演算进程
- 最后一次"再创世"不断轮回
- 死循环被称为"永劫回归"

## 3. 第十三因子：德谬歌

### 3.1 基本信息

**身份**：
- 翁法罗斯的第十三位泰坦
- 权杖δ-me13原本的演算目标：哺育"自我"的生命形态
- 被来古士从演算中剔除

**状态变化**：
- 初始：受星核污染陷入长眠
- 沉睡中：PhiLia093为其载入十二因子的数据
- 在沉睡中踏上"记忆"命途
- 苏醒：因开拓者的到来而苏醒
- 最终：通过"回望"牺牲自我，成为新世界的摇篮

### 3.2 关键特性

**路径**：创世

**原动力**：爱

**作用**：
- 观察并记录十二因子的数据
- 通过"记忆"的力量影响世界
- 最终通过"回望"实现因果闭合

### 3.3 与PhiLia093的关系

- PhiLia093是德谬歌的心识在时间起点的化身
- 在永劫回归中识破来古士的计划
- 通过"记忆"力量强行回退演算进程

## 4. 黑潮

### 4.1 本质

**来源**：
- 来古士故意引入的对抗机制
- 第二阶段演算结果的视觉化表现
- 系统高度无序状态的体现

**特性**：
- 缓慢逼近翁法罗斯边界
- 侵蚀一切生命，使其丧失理智
- 连泰坦也无法抵抗

### 4.2 作用

**在演算中的意义**：
- 对抗机制：迫使12因子进化
- 选择压力：淘汰脆弱的生命形态
- 毁灭倾向：推动系统向"毁灭"倾斜

## 5. 再创世机制

### 5.1 基本流程

1. 黄金裔击杀泰坦，回收火种
2. 十二枚火种归于创世涡心
3. 负世半神以自己的记忆再造翁法罗斯
4. 前世的半神成为现世的泰坦
5. 时间线重新开始

### 5.2 记忆传承

- 前世的负世半神保留记忆
- 用记忆重构新世界
- 其他半神转生为泰坦，失去记忆

### 5.3 永劫回归

**触发条件**：
- NeiKos496和PhiLia093识破计划
- 通过"记忆"力量强行回退演算进程

**表现**：
- 最后一次"再创世"不断重复
- 循环次数：33550336次
- 目的：阻止"铁墓"彻底诞生
